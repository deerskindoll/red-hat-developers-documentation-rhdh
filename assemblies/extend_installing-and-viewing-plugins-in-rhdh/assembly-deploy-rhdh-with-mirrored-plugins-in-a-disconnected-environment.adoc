= Deploy {product} with mirrored plugins in a disconnected environment

:context: deploy-rhdh-with-mirrored-plugins-in-a-disconnected-environment



[role="_abstract"]
After mirroring completes, the script generates a `rhdh-plugin-mirroring-summary.txt` file containing mappings of the plugin catalog index image (if specified) and all mirrored plugins.

To use the mirrored plugins in {product}, you must configure container registry mirroring so that {product} transparently pulls plugins from your mirror registry instead of the original source registries. This is done by mounting a custom `registries.conf` file and an optional `policy.json` file as extra files into the {product} init container.
