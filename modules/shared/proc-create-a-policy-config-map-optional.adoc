= Create a policy config map (Optional)

[role="_abstract"]
Depending on your target mirror registry, you might need to configure a policy file. For production environments, you should use a more restrictive policy that verifies image signatures.

.Procedure
* Create a config map containing the policy configuration:
+
[source,yaml]
----
apiVersion: v1
kind: ConfigMap
metadata:
  name: my-mirror-policy
data:
  policy.json: |
    {
      "transports": {
        "docker": {
          "<registry>/<namespace>": [
            {
              "type": "signedBy",
              "keyType": "GPGKeys",
              "keyPath": "<path>/key.gpg"
            }
          ]
        }
      }
    }
----
